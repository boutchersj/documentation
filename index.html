Introduction

Flexbox is the commonly-used name for the CSS Flexible Box Layout Module, a layout model for displaying items in a single dimension â€” as a row or as a column. In the specification, flexbox is described as a layout model for user interface design. The key feature of flexbox is the fact that items in a flex layout can grow and shrink. Space can be assigned to the items themselves, or distributed between or around the items. Flexbox also enables alignment of items on the main or cross axis, thus providing a high level of control over the size and alignment of a group of items.

Prior Knowledge

It is helpful before beginning to work with CSS Flexbox that you have a solid knowledge of pure CSS and HTML. While not necessary, it gives you a more well-rounded perspective with which to tackle any bugs that may arise in your code while learning Flexbox.

align-content

The CSS align-content property defines how the browser distributes space between and around content items along the cross-axis of their container, which is serving as a flexbox container. This property has no effect on single line flexible boxes.

The following are possible values that are widely supported on major browsers:

  • flex-start
  • center
  • flex-end
  • space-around
  • space-between
There are other values like "space-evenly" that are supported by some but not all browsers, so use those at your own discretion.

The "space-around" value aligns the boxes on the vertical axis because the default direction of the container is "row". This value creates space between the items and between the "bookend" items and the edges of the container. This as opposed to "space-between," which only creates space between the items and not at the edges.
